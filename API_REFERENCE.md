# SMS Transaction Analyzer - API Reference\n\nComplete reference for all services, repositories, and data access methods.\n\n## Core Services\n\n### SmsParsingService\n\nLocation: `lib/core/services/sms_parsing_service.dart`\n\n#### Methods\n\n##### parseSms(String messageBody, String sender) → SmsTransactionEntity?\nParse an SMS message and extract transaction details.\n\n**Parameters:**\n- `messageBody` (String): Full SMS text\n- `sender` (String): Message sender\n\n**Returns:** `SmsTransactionEntity?` - Parsed transaction or null if not a transaction SMS\n\n**Example:**\n```dart\nconst messageBody = 'You have transferred KES 1,000.00 to John. Balance: KES 5,000';\nconst sender = 'MyBank';\nfinal transaction = smsParsingService.parseSms(messageBody, sender);\n```\n\n### SmsReadingService\n\nLocation: `lib/core/services/sms_reading_service.dart`\n\n#### Methods\n\n##### getAllSms() → Future<List<RawSmsMessage>>\nRetrieve all SMS messages from device.\n\n**Returns:** `Future<List<RawSmsMessage>>` - All SMS messages\n\n**Throws:** `SmsReadingException`\n\n**Example:**\n```dart\nfinal allMessages = await smsReadingService.getAllSms();\n```\n\n##### getSmsInDateRange(DateTime startDate, DateTime endDate) → Future<List<RawSmsMessage>>\nGet SMS messages within a date range.\n\n**Parameters:**\n- `startDate` (DateTime): Start date (inclusive)\n- `endDate` (DateTime): End date (inclusive)\n\n**Returns:** `Future<List<RawSmsMessage>>` - Filtered messages\n\n**Example:**\n```dart\nfinal startDate = DateTime(2024, 5, 1);\nfinal endDate = DateTime(2024, 5, 31);\nfinal messages = await smsReadingService.getSmsInDateRange(startDate, endDate);\n```\n\n##### getSmsFromSender(String sender) → Future<List<RawSmsMessage>>\nGet SMS messages from a specific sender.\n\n**Parameters:**\n- `sender` (String): Sender name/number\n\n**Returns:** `Future<List<RawSmsMessage>>` - Messages from sender\n\n**Example:**\n```dart\nfinal bankMessages = await smsReadingService.getSmsFromSender('MyBank');\n```\n\n##### filterTransactionSms(List<RawSmsMessage> messages) → List<RawSmsMessage>\nFilter messages to find transaction-related SMS.\n\n**Parameters:**\n- `messages` (List<RawSmsMessage>): Messages to filter\n\n**Returns:** `List<RawSmsMessage>` - Transaction messages only\n\n**Example:**\n```dart\nfinal transactionMessages = smsReadingService.filterTransactionSms(allMessages);\n```\n\n### PermissionService\n\nLocation: `lib/core/services/permission_service.dart`\n\n#### Methods\n\n##### requestSmsPermission() → Future<bool>\nRequest SMS reading permission from user.\n\n**Returns:** `Future<bool>` - True if granted, false otherwise\n\n**Throws:** `SmsPermissionException`\n\n**Example:**\n```dart\nfinal granted = await permissionService.requestSmsPermission();\nif (granted) {\n  // Permission granted\n}\n```\n\n##### isSmsPermissionGranted() → Future<bool>\nCheck if SMS permission is already granted.\n\n**Returns:** `Future<bool>` - Permission status\n\n**Example:**\n```dart\nif (await permissionService.isSmsPermissionGranted()) {\n  // Can access SMS\n}\n```\n\n##### isSmsPermissionPermanentlyDenied() → Future<bool>\nCheck if permission is permanently denied.\n\n**Returns:** `Future<bool>` - True if permanently denied\n\n**Example:**\n```dart\nif (await permissionService.isSmsPermissionPermanentlyDenied()) {\n  // Show user to go to settings\n}\n```\n\n## Domain Layer\n\n### Use Cases\n\nLocation: GetTransactionsByCategoryUseCase\n\n##### call(String category) → Future<List<SmsTransactionEntity>>\nFetch transactions in specific category.\n\n**Parameters:**\n- `category` (String): Category transactions at once.\n\n**Parameters:**\n- `transactions` (List<SmsTransactionEntity>): Transactions to add\n\n**Example:**\n```dart\nfinal useCase = AddMultipleTransactionsUseCase(repository);\nawait useCase([transaction1, transaction2, transaction3]);\n```\n\n#### DeleteTransactionUseCase\n\n##### call(String id) → Future<void>\nDelete a transaction by ID.\n\n**Parameters:**\n- `id` (String): Transaction ID\n\n**Example:**\n```dart\nfinal useCase = DeleteTransactionUseCase(repository);\nawait useCase('transaction_id');\n```\n\n#### GetStatisticsUseCase\n\n##### call(DateTime startDate, DateTime endDate) → Future<Map<String, dynamic>>\nGet statistics for date range.\n\n**Parameters:**\n- `startDate` (DateTime): Period start\n- `endDate` (DateTime): Period end\n\n**Returns:** Map with:\n- `totalIncome` (double)\n- `totalExpenses` (double)\n- `savings` (double)\n- `transactionCount` (int)\n\n**Example:**\n```dart\nfinal smsReadingService.getAllSms();\nfinal transactionMessages = smsReadingService.filterTransactionSms(allMessages);\n\nfinal smsParsingService = SmsParsingService();\nfinal transactions = <SmsTransactionEntity>[];\n\nfor (final message in transactionMessages) {\n  final transaction = smsParsingService.parseSms(\n    message.messageBody,\n    message.sender,\n  );\n  if (transaction != null) {\n    transactions.add(transaction);\n  }\n}\n```\n\n### Adding transactions to database\n\n```dart\nref.read(addMultipleTransactionsUseCaseProvider).call(transactions);\n```\n\n### Getting statistics\n\n```dart\nfinal statistics = await ref.read(getStatisticsUseCaseProvider).call(\n  DateTime(2024, 5, 1),\n  DateTime(2024, 5, 31),\n);\n```\n\n### Searching transactions\n\n```dart\nref.read(searchQueryProvider.notifier).state = '1000';\nfinal results = await ref.watch(searchedTransactionsProvider(query: '1000').future);\n```\n\n## Notes\n\n- All methods are async unless otherwise specified\n- Null safety is enforced throughout\n- All entities are immutable (via Freezed)\n- Database operations are atomic where possible\n- Search is case-insensitive\n- Pagination is 1-based\n
